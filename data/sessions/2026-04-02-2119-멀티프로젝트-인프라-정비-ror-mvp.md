## 배운 점 (TIL)

- git fetch refspec이 특정 태그만 가리키면 모든 브랜치 fetch가 안 됨 (`+refs/heads/*:refs/remotes/origin/*`로 복구)
- korean-law-mcp 원격 서버도 API 키가 필요함 (README와 다름)
- MCP SDK v1.29.0은 SSEClientTransport → StreamableHTTPClientTransport로 전환됨
- LaunchD exit 255 = bash에서 SSH가 반환하는 -1 → `set -o pipefail`과 결합시 치명적
- Fly.io 앱이 Cloudflare 뒤에 있으면 curl 기본 UA는 200이지만 모바일 UA는 406 가능
