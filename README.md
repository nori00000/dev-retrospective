# Dev Retrospective

개발 세션 회고 아카이브. 작업 내역, 의사결정, 교훈을 체계적으로 기록합니다.

## 구조

```
retrospectives/     # 수동 작성 회고 (연/월별)
  2026/03/          # YYYY-MM-DD-주제.md 형식
data/sessions/      # 자동 세션 로그 (Claude Code 생성)
data/reviews/       # AI 보강 리뷰 (daily/weekly/monthly)
templates/          # 회고 작성 템플릿
```

## 작성 규칙

- 파일명: `YYYY-MM-DD-주제-키워드.md`
- 위치: `retrospectives/YYYY/MM/`
- 템플릿: `templates/session-retrospective-template.md` 참고
- 필수 섹션: 메타데이터, 세션 요약, 작업 내역, 의사결정, 교훈, Follow-up
